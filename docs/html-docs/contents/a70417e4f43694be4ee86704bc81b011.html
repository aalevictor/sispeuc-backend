enderecos

ERDEntity
SMUL/SISPEUC :: Modelo de Persistência de Dados :: enderecos

Description

Tabela criada exclusivamente para registrar os endereços dos proprietários presentes primariamente na matrícula de imóvel dos registros notoriais. Usualmente são endereços diferentes dos registros municipais de SF. A regra de negócio demanda múltiplos registros de endereço, afim de que as vistorias e notificações sejam bem sucedidas.
