ERDDataModel
SMUL/SISPEUC :: Modelo de Persistência de Dados

Description

2024-05-15 Esse modelo proposto é a pedra fundamental para o modelo de persistência de dados do "Novo SISPEUC". Para operacionalização do mesmo, será utilizado o Object Relational Model (ORM) Prisma 5.x junto com PostgreSQL 16.x.

Referências: Prisma: https://www.prisma.io PostgreSQL: https://www.postgresql.org
