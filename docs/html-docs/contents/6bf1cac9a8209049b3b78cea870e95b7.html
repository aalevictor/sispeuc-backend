imoveis

ERDEntity
SMUL/SISPEUC :: Modelo de Persistência de Dados :: imoveis

Description

Tabela criada exclusivamente para registrar os imóveis e seus dados relevantes, como setor quadra lote e endereço. Usualmente são endereços diferentes dos registros dos serviços notoriais. A regra de negócio demanda múltiplos registros de endereço, afim de que as vistorias e notificações sejam bem sucedidas.
