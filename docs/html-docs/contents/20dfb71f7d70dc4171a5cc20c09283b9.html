geoEpsg

ERDColumn
SMUL/SISPEUC :: Modelo de Persistência de Dados :: denuncias :: geoEpsg

Description

Código único amplamente utilizado para definir datum e projeção (quando houver) de dados espaciais. Consultar epsg.io Verificar que datum e projeção a biblioteca front-end e o banco de dados utilizará. Exemplos Postgis: ALTER TABLE locations ADD COLUMN geoEPSG geometry(Point, 4326); UPDATE locations SET geoEPSG = STSetSRID(STMakePoint(longitude, latitude), 4326);
